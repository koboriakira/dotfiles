# difit — ブラウザでdiffを確認

ユーザーが明示的にdifitによるdiff確認を依頼したときにのみ使用する。
実装完了後に自動でdifitを実行してはいけない。

## コマンド

| ユースケース | コマンド |
|---|---|
| HEADコミットを確認 | `npx difit` |
| コミット前の変更を確認 | `npx difit .` |
| ステージ済みの変更のみ確認 | `npx difit staged` |
| 特定ブランチとの差分を確認 | `npx difit @ main` |

## 手順

1. git status で現在の状態を確認する
2. 状況に応じて適切なコマンドを選択する
   - コミット済みの変更を見たい場合: `npx difit`
   - 未コミットの変更がある場合: `npx difit .`
   - ステージ済みのみ見たい場合: `npx difit staged`
   - ブランチ比較の場合: `npx difit @ <ブランチ名>`
3. difitを実行し、ユーザーがブラウザで確認を終えるのを待つ
4. stdoutにレビューコメントが返された場合、その内容に対応して作業を続行する
